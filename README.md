# ShahedBookShop 📚
**پلتفرم جامع فروش و تبادل کتاب دانشجویی دانشگاه شاهد**

این پروژه یک سیستم تحت وب مبتنی بر فریم‌ورک جنگو (Django) است که با هدف تسهیل خرید، فروش و تبادل کتاب‌های درسی و غیردرسی میان دانشجویان دانشگاه شاهد طراحی و پیاده‌سازی شده است. این سامانه علاوه بر فروشگاه اصلی، دارای یک ماژول اختصاصی برای ثبت درخواست فروش کتاب‌های دست‌دوم توسط دانشجویان می‌باشد.

---

## 🌟 قابلیت‌های کلیدی (Features)

- **مدیریت فروشگاه و کاتالوگ:** نمایش کتاب‌ها بر اساس دسته‌بندی، نویسنده و جدیدترین‌ها.
- **سیستم سبد خرید و سفارشات:** امکان افزودن کتاب به سبد خرید، محاسبه مجموع قیمت و ثبت نهایی سفارش.
- **پلتفرم تبادل کتاب (C2C):** امکان ثبت درخواست فروش کتاب کارکرده توسط دانشجویان همراه با آپلود تصویر جلد و تعیین قیمت.
- **گردش کار (Workflow) درخواست‌های فروش:** بررسی درخواست‌های فروش دانشجویان توسط مدیریت (تأیید/رد) و تبدیل خودکار درخواست‌های تأیید شده به محصولات فروشگاه.
- **جستجوی پیشرفته:** امکان جستجوی سریع کتاب‌ها بر اساس نام.
- **احراز هویت کاربران:** سیستم ثبت‌نام و ورود یکپارچه برای دانشجویان.
- **پنل مدیریت (Admin Panel):** مدیریت کامل کاربران، کتاب‌ها، دسته‌بندی‌ها، نویسندگان، اسلایدرها و فاکتورها.

## 🛠 تکنولوژی‌های استفاده شده (Tech Stack)

**بک‌اند (Back-end):**
- Python (پایتون)
- Django (فریم‌ورک جنگو)
- SQLite (پایگاه داده پیش‌فرض - قابل ارتقا به PostgreSQL/MySQL)

**فرانت‌اند (Front-end):**
- HTML5 / CSS3
- JavaScript / jQuery
- Bootstrap (برای طراحی ریسپانسیو)
- Slick Slider / SKD Slider (برای بنرها و اسلایدرها)

## 📁 ساختار پروژه‌ (Project Structure)

این پروژه از چندین اپلیکیشن (App) مجزا در جنگو تشکیل شده است تا معماری ماژولار حفظ شود:

- `store/`: هسته اصلی فروشگاه شامل مدل‌های کتاب، دسته‌بندی، نویسنده و صفحات اصلی سایت.
- `cart/`: مدیریت لاجیک سبد خرید با استفاده از Session ها.
- `order/`: مدیریت فاکتورها، تراکنش‌ها و اقلام خریداری شده.
- `sell_request/`: ماژول اختصاصی ثبت و پیگیری درخواست فروش کتاب‌های دست دوم دانشجویان.
- `search/`: کنترلرهای مربوط به جستجوی کتاب‌ها.
- `bookstore/`: پوشه تنظیمات اصلی (Settings) پروژه جنگو.
